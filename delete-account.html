MyShop Diary 계정 삭제 안내

⚠️ 중요한 안내사항
계정을 삭제하면 모든 데이터가 영구적으로 삭제되며 복구할 수 없습니다. 삭제 전에 중요한 데이터를 백업해 주세요.

1. 계정 삭제 방법

  1. 앱 내에서 직접 삭제 (권장)

    단계별 안내:

    1. MyShop Diary 앱 실행
    2. 하단 메뉴에서 '설정' 탭 선택
    3. '계정 정보' 섹션에서 '계정 관리' 선택
    4. '계정 삭제' 버튼 선택
    5. 삭제 사유 선택 및 확인
    6. 최종 확인 후 '계정 삭제' 실행
    처리 시간: 즉시 처리 (Firebase Authentication에서 계정 삭제)
  2. 이메일을 통한 삭제 요청

    이메일 발송 정보:

    • 받는 사람: privacy@myshop-diary.com
    • 제목: [계정삭제] MyShop Diary 계정 삭제 요청
    • 내용 포함사항:
      • 등록된 이메일 주소
      • 삭제 요청 사유
      • 본인 확인을 위한 추가 정보 (필요시)
    처리 시간: 영업일 기준 3일 이내 처리 완료
  3. 고객 지원 센터 문의

    문의 방법:

    • 앱 내 '고객 지원' 메뉴 이용
    • 이메일: support@myshop-diary.com
    • 문의 시 본인 확인 절차 필요
    처리 시간: 영업일 기준 1-2일 이내 안내 후 처리

2. 삭제되는 데이터

데이터 유형 삭제 내용 삭제 시점
계정 정보 이메일 주소, 로그인 정보 즉시
사업자 정보 사업체명, 사업자명, 주소, 전화번호, 사업자등록번호 등 즉시
고객 데이터 고객명, 연락처, 생년월일, 메모 등 모든 고객 정보 즉시
예약 데이터 모든 예약 내역, 예약 상태, 결제 정보 즉시
매출 데이터 매출 통계, 결제 내역, 수익 분석 데이터 즉시
서비스 데이터 등록된 서비스, 가격, 카테고리 정보 즉시
멤버십 데이터 멤버십 정보, 구매 내역, 사용 내역 즉시
앱 사용 기록 로그인 기록, 앱 사용 패턴, 성능 로그 즉시

3. 삭제 불가능한 데이터

법적 보관 의무가 있는 데이터

다음 데이터는 관련 법령에 따라 일정 기간 보관됩니다:

  • 세금계산서 관련 기록: 5년 (부가가치세법)
  • 전자상거래 기록: 5년 (전자상거래법)
  • 부정이용 기록: 1년 (개인정보보호법)
  • 서비스 이용 기록: 3개월 (통신비밀보호법)

보관 방식: 개인식별이 불가능한 형태로 암호화하여 별도 저장

4. 계정 삭제 전 확인사항

삭제 전 반드시 확인하세요!

  • 데이터 백업: 중요한 고객 정보, 예약 내역, 매출 데이터 백업
  • 미처리 예약: 진행 중인 예약이 있는지 확인
  • 정산 완료: 미정산 금액이 있는지 확인
  • 멤버십 처리: 고객 멤버십 잔여 횟수 처리 여부 확인
  • 복구 불가: 삭제 후에는 어떤 데이터도 복구할 수 없음을 인지

5. 부분 데이터 삭제

전체 계정 삭제가 아닌 부분적인 데이터만 삭제하고 싶은 경우:

5.1 앱 내에서 직접 삭제 가능한 데이터

  • 개별 고객 정보
  • 특정 예약 내역
  • 서비스 항목
  • 멤버십 정보

5.2 삭제 방법

  1. 해당 데이터가 있는 화면으로 이동
  2. 삭제하고 싶은 항목 선택
  3. '삭제' 버튼 선택
  4. 삭제 확인

6. 계정 삭제 후 안내

6.1 삭제 완료 통지

  • 계정 삭제 완료 시 등록된 이메일로 확인 메일 발송
  • Firebase Authentication에서 계정 정보 즉시 삭제
  • Cloud Firestore의 모든 개인 데이터 즉시 삭제

6.2 재가입 안내

  • 동일한 이메일로 재가입 가능
  • 기존 데이터는 복구되지 않음
  • 새로운 계정으로 처음부터 시작

7. 삭제 철회 및 유예

삭제 유예 기간
이메일로 삭제 요청한 경우, 처리 전까지는 철회 가능합니다. 단, 앱 내에서 직접 삭제한 경우에는 즉시 처리되어 철회가 불가능합니다.

9. 자주 묻는 질문 (FAQ)

Q1. 계정 삭제 후 복구가 가능한가요?

A: 아니요. 계정 삭제 후에는 모든 데이터가 영구적으로 삭제되어 복구가 불가능합니다.

Q2. 일부 데이터만 삭제할 수 있나요?

A: 네. 앱 내에서 고객, 예약, 서비스 등의 개별 데이터를 삭제할 수 있습니다.

Q3. 삭제 처리에 얼마나 걸리나요?

A: 앱 내 삭제는 즉시, 이메일 요청은 영업일 기준 3일 이내 처리됩니다.

Q4. 동일한 이메일로 재가입이 가능한가요?

A: 네. 계정 삭제 후 동일한 이메일로 재가입이 가능하지만, 기존 데이터는 복구되지 않습니다.

Q5. 법정 보관 의무 데이터는 언제 삭제되나요?

A: 관련 법령에서 정한 보관 기간이 만료되면 자동으로 삭제됩니다.
